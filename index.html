Forever at War

Forever at War is an experimental 2D strategy game made with Godot.

Please make sure you have the latest version of WebGL enabled. The game will take some time to load relevant assets depending on your internet connection speed. This game is not intended to be played on mobile devices.

To play the web build, go to /web/ForeverAtWar.html

The game might lag on slower computers. If that's the case, there is also a Windows build.

To download the windows build, go to the /win/ folder and download the two files.

Please keep in mind that this game is an experiment. Expect bugs, glitches and balancing issues and AI issues in your playthrough. Other than that, have fun!
